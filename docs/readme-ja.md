# FGO周回ソルバー

FGOの集めたい素材の数から、どのクエストを何周するのが最も効率的かを求めるサイトです。

## 使い方

### 育成素材計算機

1. 全サーヴァント共通の強化設定（強化対象・育成レベル）を行います。
2. 育成したいサーヴァントを選択し、個別設定を行います。
3. 計算ボタンを押すと必要素材の一覧が表示されます。
4. アイテムの所持数を入力すると不足数が計算されます。
5. 「周回数を求める」ボタンを押すと周回ソルバーへ移ります。

### 周回ソルバー

1. 集めたいアイテムと個数を入力します。
2. 周回対象に含めるクエストを選択します。
3. 計算ボタンを押すと計算が始まります（数秒かかります）。
4. 結果は **消費AP 最小** と **周回数 最小** の2パターンをタブで切り替えて確認できます。

### 計算履歴

過去の計算結果を一覧で確認できます。消費AP推移・周回数推移のグラフも表示されます。

## 機能

- 入力フォームの内容は自動的にブラウザに保存されます。
- 計算結果のページから計算結果をX（旧Twitter）でポストして共有できます。
- Googleアカウントでログインするとデータをクラウドに保存し、複数デバイス間で同期できます。

## データの取り扱いについて

- 入力された値の送受信は暗号化されています。
- 送信された値は必要周回数の計算に使用します。
