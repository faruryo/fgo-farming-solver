## 2026-06-16 — 所持アイテムデータの自動取り込みを当面見送る

**ステータス**: 見送り（Deferred）

### 背景
競合（特に Chaldea）は、所持素材を手入力せずに取り込む機能を持つ。これが乗り換え障壁になりうるため、本プロダクトでも実現可能か検討した。検討した方式は次の2つ。

1. **スクリーンショット OCR** — 所持アイテム一覧のスクショを解析（アイコンの perceptual hash 照合 + 数字認識）。
2. **HTTPS スニッフィング** — ゲームログイン時の通信を傍受し、サーバーが返す JSON を丸ごと取得。

### 決定
**どちらも当面実装しない。** データ自動取り込み機能はスコープ外とする。

### 理由
- **スクショ OCR はページング地獄**: FGO の素材一覧は1画面に少数しか表示されず、全素材を撮るには何枚もスクショして連続アップロードが必要。手入力より面倒になりかねず本末転倒。先行する Chaldea もスクショ方式を保守終了している（参考: https://docs.chaldea.center/guide/import_data ）。
- **HTTPS スニッフィングは JP iPhone 非対応**: Chaldea のドキュメント（ https://docs.chaldea.center/guide/import_https/ ）の通り JP/iPhone はサポート外で、開発者本人の環境で使えない。自分が使えない機能を継続保守するのは現実的でない。
- **コア体験のレバーではない**: 本プロダクトは既にクラウド同期で所持数を永続化している。所持数の入力は「毎回」ではなく「変わった分だけ更新」であり、手入力の苦痛は実質ワンタイムに償却される。自動取り込みは乗り換え障壁を下げる効果はあっても、コア体験を変える機能ではない。
- **保守コストが高い**: 認識系はゲーム UI 更新・新素材追加・新端末のアスペクト比で壊れる。継続メンテの負担が大きい。

### 再検討の条件
次のいずれかが成立したら再検討する。
- 他ツール（Chaldea / FGO Manager 等）のエクスポートファイル取り込みのような、**低コストで「手入力したくない」を満たす手段**を入れたくなったとき（OCR より圧倒的に安い）。
- マルチモーダル LLM による画像直読みの**精度・コスト・レイテンシが無料運用と両立**するレベルに改善したとき。
- **JP iPhone でも使える**スニッフィング/連携手段が確立したとき。
