# No shadcn/ui Migration

This project uses **Chakra UI v2** as its UI framework (see AGENTS.md).

Do NOT suggest migrating to shadcn/ui + Tailwind CSS.
Do NOT load the `shadcn` skill.
Do NOT replace Chakra UI components with shadcn/ui components.

The `react-best-practices` skill's shadcn upgrade suggestion does not apply here.
