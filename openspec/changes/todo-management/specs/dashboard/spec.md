## ADDED Requirements

### Requirement: ダッシュボードでの TODO 進捗表示 (Dashboard TODO Widget)
システムは、ダッシュボードのトップページ（`/`）の上部（上段グリッドの前または内側）に、期限切れ間近の未完了 TODO タスクを表示する「期限間近のタスク」ウィジェットを表示しなければならない (SHALL)。未完了の自動生成またはカスタム TODO が存在しない場合は、ウィジェット自体を非表示とする。

#### Scenario: 期限間近の TODO 表示
- **WHEN** 未完了のデイリーミッション、ウィークリーミッション、またはイベント交換 TODO が存在し、それらの期限が閾値（デイリー: 24時間以内、ウィークリー/イベント: 48時間以内）未満のとき
- **THEN** ダッシュボード上部に「期限間近のタスク」ウィジェットが表示され、期限が近い順にタスク名と残り時間のカウントダウンが表示される。

#### Scenario: TODO 完了時のウィジェットの自動更新
- **WHEN** ユーザーがダッシュボードの TODO ウィジェット上でタスクのチェックボックスをオン（完了）にしたとき
- **THEN** そのタスクはウィジェットのリストから即座に削除（または完了表示）され、すべての表示対象タスクが完了状態になった場合はウィジェットが非表示になる。
