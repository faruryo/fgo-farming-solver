## Why

現在のトップページは機能へのリンクが並んでいるだけのシンプルな状態で、ユーザーから「寂しい」という評価を受けています。高度な計算アルゴリズムや既存のセーブ機能を活かしきれておらず、ユーザーが「今何をすべきか」を直感的に把握できる、情報密度の高いダッシュボードへと進化させる必要があります。

## What Changes

- **ヒーローセクションのダッシュボード化**: 
    - Atlas Academy API から取得した「開催中イベント」「開催中ガチャ」の自動表示。
    - イベント終了およびアイテム交換期限のカウントダウン提示。
- **パーソナル目標の可視化**: 
    - 保存された「所持素材」「目標サーヴァント」に基づき、育成進捗を円グラフやプログレスバーで視覚化。
- **インテリジェント・ナビゲーション**: 
    - 現在の不足素材がドロップするイベントクエストやフリークエストを「推奨」として提示。
- **情報アーキテクチャの再設計**: 
    - プログレッシブ・ディクロージャー（段階的開示）を採用し、複雑な設定フォームをウィザード形式に整理。
- **ビジュアルデザインの刷新**: 
    - FGOの世界観を反映した「Glassmorphism」や「発光エフェクト」を活用したリッチなUIを提供。

## Capabilities

### New Capabilities
- `dashboard`: ユーザーの進捗、開催中イベント、推奨クエスト、ガチャ情報を統合的に表示する中央管理画面。
- `progressive-disclosure-form`: 複雑なソルバー設定を段階的に開示し、認知負荷を軽減する入力インターフェース。

### Modified Capabilities
- `solver`: 特定クエストのドロップ品表示やキャンペーン情報の連携。

## Impact

- **フロントエンド**: `app/page.tsx` の全面刷新。多数のダッシュボードコンポーネント新設。
- **データ層**: `updater-worker` による Atlas Academy API (event, gacha) の定期取得とKV保存。
- **UI**: `globals.css` のデザインシステム拡張、Recharts の導入。
