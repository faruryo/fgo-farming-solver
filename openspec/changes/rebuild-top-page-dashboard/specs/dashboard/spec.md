# Capability: Dashboard

## Overview
ユーザーがログインした直後に「現在のゲーム状況」と「自分の育成状況」をひと目で把握し、次に何をすべきか（どのクエストを周回すべきか）を導き出すための中枢機能。

## Requirements

### 1. 開催中イベント・キャンペーン表示 (Automated)
- **イベントバナー**: `nice_event.json` から取得したバナー画像を表示。
- **カウントダウン**: 
    - クエスト終了 (`endedAt`) までの残り日数。
    - アイテム交換所終了 (`shopFinishedAt`) までの残り日数。
- **ドロップアイテム表示**: 現在開放されているイベントクエストのドロップ品（品目のみ）をアイコンで一覧表示。
- **キャンペーン情報**: AP消費1/2、大成功率アップ等のキャンペーンが現在有効な場合にバッジ表示。

### 2. 現在のピックアップガチャ表示 (Automated)
- **ガチャバナー**: `nice_gacha.json` から現在開催中の全バナーをスライダーまたはリストで表示。
- **ピックアップ対象**: バナーに関連付けられたピックアップサーヴァントの顔アイコン（`face`）を表示。
- **終了期限**: 各ガチャの終了日時までのカウントダウン。

### 3. パーソナライズ進捗表示 (Strategic)
- **進捗グラフ**: ユーザーが設定した「目標サーヴァント」の全スキル・霊基再臨の達成度を Recharts 等のドーナツチャートで表示。
- **不足素材リスト**: 目標達成に最もクリティカルな不足素材（例：あと5個で完成するもの等）を優先的に表示。
- **推奨クエスト**: 不足素材がドロップするイベントクエスト、または最高効率のフリークエストを提示。

### 4. クイックアクション (Operational)
- 複雑な設定ページへ飛ばずとも、トップページから「素材計算」「履歴確認」等の主要機能へ即座にアクセスできるナビゲーション。
- モバイル利用を考慮した、親指で操作しやすい Sticky Footer。

## User Stories
- **イベント走破型マスター**: 「アンティオキアのレイドで証が落ちることをトップページで知り、交換所の期限も把握した上で、即座に周回を開始したい」
- **目標達成型マスター**: 「モルガンの全スキルマまであと素材が何個必要かを確認し、推奨されたクエストに向かいたい」
- **新米マスター**: 「今どのガチャが開催されていて、誰がピックアップされているかを画像付きで確認したい」
